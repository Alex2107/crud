Приветствую!
Пошаговая инструкция запуска приложения:

1)Запустить скрипт crud->src->main->resources->create_db_and_tbl.sql.
Он создаст схему и таблицу

2)Запустить скрипт crud->src->main->resources->insert_data.sql.
Он наполнит таблицу живительными данными

3)Скомпилить и запустить приложение.

4)Пойти в браузере по ссылке http://localhost:8080/ 

5)Узреть список книг. При нажатии ссылки Title появится полная инфа о книге
на новой странице. Для добавления книги внести данные в поля после надписи Add a Book
и нажать кнопку Add Book. Для редактирования нажать ссылку Edit возле книги,
поля заполнятся автоматически и будут доступны для редактирования.
По окончанию редактирования нажать Edit Book. Ссылка Delete удалит запись.
